Swagger 로그아웃 /** Swagger 링크 (/swagger-ui/index.html): 이것은 SpringDoc/Swagger UI가 자동으로 제공하는 경로입니다. AuthController.java 코드에서 @SecurityScheme와 @Operation(security = @SecurityRequirement(name = "bearerAuth")) 어노테이션을 사용한 것을 볼 수 있는데, 이는 Swagger 문서화를 위한 설정입니다. SpringDoc 라이브러리가 프로젝트에 추가되면 자동으로 /swagger-ui/index.html 경로에 API 문서 페이지를 생성합니다. 따라서 명시적인 컨트롤러 매핑 없이도 이 경로가 작동합니다. 로그아웃 링크 (/logout): 이것은 Spring Security의 기본 제공 엔드포인트입니다. Spring Security가 프로젝트에 구성되어 있으면, 기본적으로 /logout 경로를 제공하며 사용자의 세션을 종료하고 인증 정보를 제거합니다. 직접 컨트롤러에 이 경로를 정의하지 않아도 Spring Security 설정에 의해 자동으로 처리됩니다. **/
